# PASS: Actuation Module

Part of the **PASS** platform (see [`../README.md`](../README.md)).

**Status:** placeholder - awaiting implementation.
**Owner:** TBD

## Scope

Assistive actuation and load-cell force/strength measurement. Covers the force/load
side of the platform (e.g. resistance and strength as force), which the knee IMU
module cannot provide on its own.

## Where things go

Put your source code, tests, firmware and docs for the actuation subsystem in this
folder. Keep it self-contained (its own `requirements.txt` / setup) so it runs
independently. See the [`knee/`](../knee/) subsystem for a reference structure.
